Bwitterkuchen
----

CUI Twitter Client

- `auth`: Generating OAuth token
- `tw`: CLI tool
- `kuchen`: CUI Twitter Client

### Get to work

```
~$ stack build
~$ mkdir token
~$ stack exec auth
(type your consumer_key, consumer_secret)
(generate OAuth token with URL printed) 
~$ stack exec kuchen 
(type your screen_name)
```
